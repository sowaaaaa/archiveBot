# 🏰 Архив Происхождения — Telegram Bot

Готический Telegram-бот для морфологического анализа внешности с экспертным разбором.

## Быстрый старт

### 1. Установка зависимостей

```bash
cd C:\Users\sowaa\archive-bot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
copy .env.example .env
```

Заполни `.env`:

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
| `ANTHROPIC_API_KEY` | API-ключ из [console.anthropic.com](https://console.anthropic.com) |
| `ADMIN_IDS` | Твой Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)) |
| `ADMIN_CHAT_ID` | ID группы/канала для уведомлений (опционально) |
| `EXPERT_PRICE` | Цена экспертного разбора в рублях |
| `CLAUDE_MODEL` | Модель: `claude-opus-4-8` (лучше) или `claude-sonnet-4-6` (быстрее) |

### 3. Приветственное изображение (опционально)

Положи изображение замка/архивариуса в:
```
assets/welcome.jpg
```
Если файл отсутствует — бот отправит текстовое приветствие.

### 4. PDF-источники для RAG (опционально)

Положи PDF-материалы для контекста анализа в:
```
data/pdfs/
```
Бот автоматически загрузит их при запуске и включит в промпты Claude.

### 5. Запуск

```bash
python run.py
```

## Пользовательский поток

```
/start
  └─► Приветствие (фото замка + текст)
       └─► [Открыть архив]
            └─► Фото анфас
                 └─► Фото профиль
                      └─► Фото в рост? (пропустить)
                           └─► Доп. данные? (возраст / нац. / происх. / антропометрия)
                                └─► Анализ (анимация загрузки)
                                     └─► Результат (в виде книги)
                                          └─► Оффер экспертного разбора
                                               └─► Заявка → уведомление администратору
```

## Команды администратора

| Команда | Действие |
|---|---|
| `/admin` | Открыть панель администратора |

В панели доступно:
- **Статистика** — пользователи, анализы, заявки
- **Пользователи** — последние 10 пользователей с деталями
- **Заявки на разбор** — список ожидающих заявок с кнопками принять/отклонить
- **Рассылка** — отправить сообщение всем пользователям
- **Редактировать тексты** — изменить ключевые тексты бота через БД

## Безопасность контента

Бот настроен на строгое соблюдение этических ограничений:
- Только морфологический/физический анализ
- Никаких политических, социальных или интеллектуальных оценок
- Никаких идей превосходства или неполноценности
- Нейтральная, научно-антропологическая терминология

## Добавление PDF-источников для RAG

1. Помести PDF-файлы в `data/pdfs/`
2. Перезапусти бота — источники загрузятся автоматически
3. Claude будет использовать эти материалы как контекст при анализе

Рекомендуется добавлять научные материалы по:
- Антропологии и морфологии лица
- Классификации фенотипов
- Краниометрии и соматологии
